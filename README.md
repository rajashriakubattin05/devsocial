CodingHub: A Social Media Platform for Developers
Project Description

CodingHub is a social media platform designed specifically for developers and learners. Users can share coding videos, upload code snippets, interact with AI for code explanations and bug detection, and connect with other developers in a distraction-free coding environment.

Features

Upload coding videos and code snippets

AI-based code validation and bug detection

Personalized feed and content recommendations

Like, comment, share, and follow other developers

Admin moderation and analytics dashboard

Tech Stack

Frontend: React / Flutter

Backend: FastAPI / Django

Database: MongoDB Atlas

AI Tools: Google Gemini, AutoML

Storage: AWS S3 / Google Cloud Storage

Authentication: JWT / OAuth

Installation / Setup

Clone the repository:

git clone <repo-link>

Navigate to the project folder:

cd CodingHub

Install backend dependencies:

pip install -r requirements.txt

Create a .env file and add database and API keys.

Run the backend server:

uvicorn main:app --reload

Run the frontend:

npm install
npm start

Or for Flutter:

flutter run
Usage

Register or login to the platform.

Upload coding videos or code snippets.

Ask AI for code explanations or bug detection.

Interact with other developers by liking, commenting, or sharing posts.

Explore personalized feed and recommended posts.
